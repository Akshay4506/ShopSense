# ShopSense

## Project Overview

**ShopSense_WHackiest** is a full-stack web application developed as part of a hackathon initiative. The project simulates a smart and interactive shopping platform that integrates a frontend user interface with a backend server to deliver dynamic content and real-time responses.

The application demonstrates how modern web technologies can be combined to create a seamless client-server architecture similar to real-world e-commerce platforms.

---

## Project Objective

The main objective of ShopSense is to:

* Build a full-stack web application
* Demonstrate frontend and backend integration
* Implement RESTful API communication
* Simulate an intelligent shopping experience
* Apply practical web development concepts in a real project

---

## What the Project Does

ShopSense provides:

* A responsive shopping interface
* Display of product listings or dynamic content
* Interaction between frontend and backend
* API-based data communication
* Real-time updates from the server

---

## Technologies Used

* **Frontend:** JavaScript / TypeScript
* **Backend:** Node.js
* **Package Manager:** npm
* **API Communication:** REST Architecture

---

## Execution

### Step 1: Clone the Repository

```bash
git clone https://github.com/Akshay4506/ShopSense_WHackiest.git
cd ShopSense_WHackiest
```

---

## Running the Backend

1. Navigate to the backend folder:

```bash
cd backend
```

2. Install dependencies (if package.json exists):

```bash
npm install
```

3. Start the backend server:

```bash
node server.js
```

The backend will start running on a local port (commonly `http://localhost:5000`).

---

## Running the Frontend

1. Open a new terminal and navigate to the frontend folder:

```bash
cd frontend
```

2. Install dependencies:

```bash
npm install
```

3. Start the frontend application:

```bash
npm run dev
```

The frontend will typically run at:

```
http://localhost:5173
```
